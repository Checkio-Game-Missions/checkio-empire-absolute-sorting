Давайте подивимося на сортування. Заданий масив з особливими правилами.

Масив (tuple) містить різні числа. Вам необхідно відсортувати їх, але впорядкувати на основі абсолютних значень у зростаючому порядку. Для прикладу, послідовність (-20, -5, 10, 15) буде відсортована таким чином (-5, 10, 15, -20). Ваша функція повинна повертати список (list) або кортеж (tuple).

Зауваження: Це завдання може бути легко розв’язане з використанням таких функцій: {% if "python 2" in interpreter|lower %} sorted і abs. Так само ви повинні спробувати застосувати параметр key для сортування. {% else %} sorted і abs. Так само ви повинні спробувати застосувати параметр key для сортування. {% endif %}

Вхідні дані: Масив чисел як кортеж (tuple).

Вbхідні дані: Список (list) або кортеж (tuple) (але не генератор) відсортований за абсолютними значеннями чисел.

Доповнення: Результат вашої функції ви побачите як список (list) на панелі виводу результатів перевірки.

Приклади: 

checkio((-20, -5, 10, 15)) == [-5, 10, 15, -20] # or (-5, 10, 15, -20)
checkio((1, 2, 3, 0)) == [0, 1, 2, 3]
checkio((-1, -2, -3, 0)) == [0, -1, -2, -3]

Як це використовується: Сортування – це складова дуже багатьох завдань, так що буде корисно зрозуміти, як її використовувати.
